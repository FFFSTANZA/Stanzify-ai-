### Writing Your Prompt

**Do**:
- Be specific about your topic
- Mention key points you want covered
- Include any data or statistics you want
- Specify your target audience
- Mention desired slide count (e.g., "10-slide presentation")

**Don't**:
- Be too vague (e.g., "make a presentation about business")
- Use overly technical jargon without context
- Request too many topics in one presentation

## ğŸ†˜ Troubleshooting

### Presentation Not Generating
- Check your internet connection
- Ensure prompt is not empty
- Try refreshing the page
- Try a shorter, simpler prompt

### Images Not Loading
- Unsplash API may be temporarily unavailable
- Fallback images will be used
- Try regenerating
- Or select "No images" option

### Slides Look Wrong
- Try a different design style
- Regenerate for fresh layout
- Download and edit markdown manually

### Math Equations Not Rendering
- Ensure proper LaTeX syntax
- Use `$` for inline math
- Use `$$` for display math
- Check for syntax errors
