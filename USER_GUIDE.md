## üéØ Overview

Stanzify is an AI-powered presentation generator that creates beautiful, professional Slidev presentations from simple text prompts. Using Groq AI with the llama-3.1-8b-instruct model, it generates production-ready slides with advanced features like Mermaid diagrams, code highlighting, and math equations.

## ‚ú® Key Features

### Single-Page Interface
- **Prompt Input**: Describe your presentation in detail (minimum 50 characters)
- **Sample Prompts**: Click pre-made examples to get started quickly
- **Collapsible Customization**: Optional design controls for color, style, images, and purpose
- **Real-time Generation**: Watch your slides appear as they're generated
- **Full Slidev Rendering**: All Slidev features displayed directly in the web app
